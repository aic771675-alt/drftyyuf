# Domain Reserved Page

Простая одностраничная страница-заглушка для зарезервированного домена.

## Особенности
- Минималистичный чёрно-белый дизайн
- Полностью на русском языке
- Адаптивная вёрстка для всех устройств
- Простой и лаконичный текст

## Структура
- `index.html` - главная страница
- `app.js` - React компонент с основным содержимым

## Редактирование
Чтобы изменить ссылку на Telegram, отредактируйте URL в `app.js`:
```
https://t.me/neurocraftsru
```
